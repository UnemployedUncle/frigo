# 요리 여정

요리여정(Parcours Cuisine)는 냉장고에 이미 있는 재료를 기준으로 식사를 계획하고 요리까지 이어지게 돕는 서비스입니다.
냉장고에 있는 재료들과 유통기한을 기반으로 레시피를 추천해주며, 사용자가 자신만의 레시피를 만들어 나가게 돕습니다.

- Parcours Cuisine : 요리 여정
- Frigo : 냉장고
- Plan Repas(한끼 계획) & Courses(장보기)
- Étapes(단계, 스탭) : 단계


## 제품 소개

Frigo는 다음 문제를 해결하는 것을 목표로 합니다.

- 냉장고 내 재료: 냉장고에 있는 재료를 제때 쓰지 못해 버리게 되는 문제
- 요리 레시피: 매번 무엇을 먹을지 고민하느라 요리 계획이 끊기는 문제
- 장보기 지원: 필요한 것보다 더 많이 장보게 되는 문제
- 요리 취미: 요리를 꾸준한 습관과 취미로 만들기 어려운 문제


## 핵심 가치

- 냉장고에 따른 레시피 추천: 새로 사기 전에 이미 가진 재료부터 활용, 최소한으로 장보는 경우, 마음껏 장보는 경우에 따라 요리를 추천
- 유통기한 중심 의사결정: 빨리 소비해야 할 재료를 우선순위에 둡니다.
- 장보기 최소화: 필요한 재료만 모아 효율적으로 구매합니다.
- 요리 습관화: 계획, 장보기, 조리, 기록이 한 흐름으로 이어지도록 설계합니다.

## 주요 기능

### 1. Parcours Cuisine - Home 화면

요리 완료 기록은 GitHub 잔디 심기와 유사한 방식으로 누적됩니다.

- 요리 잔디 심기: 일단위로 기록되며, 요리 횟수에 따라 색깔이 진하게 표현, 일주일이 한 줄로 표현
- 레시피 갤러리: 완료한 레시피 갤러리식 나열, 사진 기반 추천 레시피

Tech. 추천 레시피



### 2. Frigo - 냉장고 재료 관리 화면 (Step 2. 시각화)

사용자는 냉장고 속 재료를 말로 등록하고 상태를 확인할 수 있습니다.
재료의 남은 수량과 기간을 직관적으로 인지할 수 있게 표현하는 것이 중요합니다.

- 재료명
- 수량
- 유통기한
- 남은 일수
- 좋아요? 등 사용자 선호에 따른 정렬

Tech. 자연어 입력에 따른 재료 정확한 정리 및 손으로 쉽게 수정 가능한 UI



### 3. Plan Repas & Courses - 단일 레시피 추천 및 장보기 목록 (Step 2. 주간 레시피 추천)

Frigo는 빨리 사용해야 하는 재료에 더 높은 우선순위를 두고 레시피를 추천합니다.

- 추천 레시피

예시:
- 시금치 1일 남음
- 새우 2일 남음
- 버터 10일 남음
추천 결과는 시금치 파스타, 갈릭 쉬림프, 새우 리조또처럼 임박 재료를 먼저 활용하는 방향으로 구성됩니다.

Tech. 레시피 제안, 장보기 목록 도출(예상 가격)

#### 3.1. 일간 장보기 목록 제공 (Step 2. 주간 식단 계획에 따른 최적화)

한두개의 필수적인 재료 구매가 필요한 경우 제안합니다.

- 장보기 목록

#### 3.2. 주간 식단 계획

- 주간 레시피

단일 레시피 추천에 그치지 않고, 일주일 단위의 요리 계획을 제안합니다.
- 재료 재사용 극대화
- 유통기한 우선 반영
- 추가 장보기 최소화

예시:
- 월: 시금치 파스타
- 화: 새우 리조또
- 수: 데리야키 치킨
- 목: 버섯 리조또
- 금: 스테이크 샐러드

#### 3.2.2. 주간 장보기 최적화

- 주간 장보기

주간 식단에 필요한 부족 재료를 한 번에 모아 장보기 목록으로 정리합니다.

- 중복 재료는 합쳐서 표시
- 꼭 필요한 재료만 추출
- 냉장고 재고 사용량과 연결

목표는 장보기를 자주 하지 않고도 식단을 완성하는 것입니다.



### 4. Étapes - 전체 레시피 조망 후 쿠킹 모드 (Step 2. 쿠킹 모드)

각 레시피 시간에 따라 단계별로 수행하고 종료된 경우 간단히 넘길 수 있게 구성합니다.
전체 Workflow를 볼 수 있고 개별 단계에 들어가면 시간에 따라 흘러가게 구성되어 있습니다.

- 전체 Workflow 조회

- 조리 시간
- 재료 목록
- 필요한 도구
- 단계별 조리 순서

#### 4.2. 쿠킹 모드

실제 요리 중에는 단계별로 화면이 전환합니다.
- 단계별 수행

- 책 넘기듯 좌우로 스위핑 (좌측 이전 단계, 우측 완료)
- 타이머 지원
- 클릭 시 시간 정지
- 짧고 명확한 안내

- 나만의 레시피 추가 ~ 시간, 재료 등을 변경하면 내 이름 단 것으로 변경

Tech. 실제 소요 시간 기록, 변경한 재료 수정

## MVP 화면 구성

Frigo MVP는 아래 5개 화면을 중심으로 구성합니다.

### 1. Parcours Cuisine

- 요리 잔디 심기
- 달성한 레시피 갤러리식 나열
- 사진 기반의 추천 레시피


### 2. Frigo

- 냉장고 내 재료
- 유통기한 표시
- 정렬 기능


### 3. Plan Repas & Courses

- 추천 레시피
- 장보기 목록
- 주간 레시피, 장보기로 확장


### 4. Étapes

- 전체 Workflow 조회
- Cooking mode에서 단계별 수행
- 실제 소요 시간 기록



## MVP 범위와 기술 방향

- 사진 위주의 시각적인 서비스
- 개인화 Agent 개발을 위한 Feedback loop 구성
- 장보기 스타일, 요리 다양성(시도), 레시피 변경 등 Skills로 저장하여 공유
